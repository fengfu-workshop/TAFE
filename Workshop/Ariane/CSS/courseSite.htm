Web Development Diploma

This course covers the skills and knowledge needed to plan, design, implement, test and maintain web applications. The skills and knowledge you will learn span several areas including:

  • Web Site architecture
  • eCommerce
  • application management
  • Web Security
  • Advanced eCommerce programming

Prerequisites

Applicants will need to have completed 3756 Certificate IV in Web Site Design, or equivalent. To find out if you have the equivalent contact the college on (02) 9472 1973

Course duration

The course is offered part time and fulltime starting February 2006.

  • Fulltime: 1 year 26 hours per week
  • Part Time: 1.5 years 12 hours per week

Applications

This course is not by application. If you would like to enrol in this course please contact the course leader via email ariane.warnant@tafensw.edu.au or complete an expression of interest form on our web site http://infotech.hornsby.tafensw.edu.au/eoi.aspx
